Описание

Данная задание реализует программу на языке C++, которая:
- Создает динамический массив целых чисел указанного ползователем размера.
  
- Заполняет массив случайными числами.
  
- Вычисляет среднее значение элементов массива:
           - последовательно;
           - параллельно с использованием OpenMP.
  
- Замеряет время выполнения последовательного и параллельного алгоритмов.
- 
- Освобождает память, выделенную под массив, после завершения работы программы.

Программа позволяет сравнить скорость последовательного и параллельного высичления среднего значения.
